# Cucumber Expressions

Use Cucumber Expression placeholders to capture parameters from step text.

## Overview

Cucumber Expressions provide an expressive, readable syntax for matching step text and extracting parameters. Gherkin Testing supports the full Cucumber Expression specification including built-in types, optional text, alternation, and custom parameter types.

# Writing Feature Files

Learn the Gherkin syntax for describing application behavior in `.feature` files.

## Overview

Gherkin is a structured, human-readable language for specifying software behavior. Gherkin Testing supports the full Gherkin v6+ specification including Feature, Rule, Background, Scenario, Scenario Outline, DataTable, DocString, Tags, and Comments.

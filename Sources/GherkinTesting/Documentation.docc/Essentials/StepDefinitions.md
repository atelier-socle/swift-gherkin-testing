# Step Definitions

Define Swift handlers for each Gherkin step using macro annotations.

## Overview

Step definitions connect Gherkin steps to Swift code. Use `@Given`, `@When`, `@Then`, `@And`, and `@But` macros to annotate methods that implement each step's behavior.
